Manuel D'utilisation de l'application Marvel

1. Double-cliquer sur le fichier index.php pour lancer l'application à partir de votre navigateur web

2. Vous devriez voir apparaitre un tableau représentant 20 personnages de l'univers Marvel :
	- Le nom du personnage
	- Son portrait
	- Possibilité de l'ajouter à un favoris

3. En cliquant sur le nom d'un des personnages vous en apprendrez un peu plus sur ce dernier :
	- Une rapide description du personnage
	- Nombre de comics où le personnage apparait 
	- Les 3 1er comics où le personnage apparait

4. Vou pouvez regarder tour à tour les différentes informations sur les personnages

5. Vous pouvez ajouter vos personnages préférés à vos favoris en cliquant sur "Ajouter aux favoris".
	Attention vous ne pouvez avoir que 5 personnages favoris à la fois.

6. Pour afficher seulement vos personnages favoris, appuyer sur le bouton "Voir les favoris" se trouvant tous en bas de la page.

7. Pour revoir l'intégralités de tous les personnages, vous pouver appuyer sur le bouton "Remontrer tous les personnages".
